Our Project

Our project aims at contributing actively to the development of a Quantum Key Management System for a Quantum Key Distribution Network which is being done by IT in the scope of the Discretion and PTQCI projects.

Quantum Key Distribution (QKD) enables the negotiation of cryptographic keys in a secure manner based on quantum information theory (without relying on computational complexity to achieve its security).

A basic element of QKD is a pair of QKD modules linked by a QKD link that allows two remote parties to share secure keys, the QKD Network consists of two or more of these elements. Keys generated by the physical layer of the QKD Network need a Key Management System that will manage and provide key material to secure cryptographic applications.

The Key Management System is basically a connection between the physical layer and cryptographic applications, including at least, a way to store keys generated by QKD Modules, relay keys between the nodes of the QKD Network and supply keys to cryptographic applications upon requests from users, all in a secure manner.

The system will follow international and EU standards specified by institutions such as ETSI and ITU-T.

Main Milestones:

1º-Very basic KMS ( ITU-T Y.3803 / DISCRETION-D3.1 )

2º-Interface to the applications ( ETSI QKD GS 004/14 )

3º-Interface to the Quantum Layer ( ETSI QKD GS 004 )

4º-KMS 1.0 ( all vital functionalities )

5º-KMS 2.0
